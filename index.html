My Roblox UI Library

Library Link: https://pastebin.com/raw/UfP9DBzE

Description

This is a Roblox UI Library made to help scripters create beautiful user interfaces easily, just like OrionLib. It supports tabs, buttons, sliders (mobile friendly), and draggable/minimizable windows. Perfect for making your own custom Roblox scripts or tools!

How to Use

local lib = loadstring(game:HttpGet("https://pastebin.com/raw/UfP9DBzE"))()
local win = lib:CreateWindow("My Script UI")

Create a Tab

local tab = win:CreateTab("Main")

Add a Button

tab:AddButton("Click Me", function()
    print("Hello!")
end)

Add a Slider

tab:AddSlider("WalkSpeed", 0, 100, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

Full Example

local lib = loadstring(game:HttpGet("https://pastebin.com/raw/UfP9DBzE"))()
local win = lib:CreateWindow("Example UI")

local main = win:CreateTab("Main")

main:AddButton("Say Hello", function()
    print("Hello from UI!")
end)

main:AddSlider("JumpPower", 0, 300, 50, function(value)
    game.Players.LocalPlayer.Character.Humanoid.JumpPower = value
end)
